## What this GitHub is for

This account is organized as a proof-of-work layer for the portfolio site.

- Public repos are for inspectable sites, demos, and portfolio surfaces.
- Private repos hold active product work, client-sensitive experiments, and source that is better shown as case studies.
- Forks are upstream references, experiments, or dependency research, not the main portfolio signal.
